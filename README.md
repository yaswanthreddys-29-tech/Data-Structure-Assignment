## Assignment Description
This repository contains the C programs for Programming Assignment 2, focusing on the applications of Stacks and Queues.

### Programs Included:
1. **Reverse a String:** Using a stack to reverse input text.
2. **Balanced Parentheses:** Using a stack to check if an expression is mathematically valid.
3. **Next Greater Element:** Finding the next larger value for array elements using a stack.
4. **Printer Queue Simulation:** A simulation of a document handling system using a queue.
5. **Circular Queue:** Implementation of a menu-driven circular queue using an array.
